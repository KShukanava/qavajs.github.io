@qavajs/steps-playwright

Step library to work with playwright using DSL page object

Installationâ€‹

npm install @qavajs/steps-playwright

Configâ€‹

const App = require('./page_object');
module.exports = {
    default: {
        require: [
            'node_modules/@qavajs/steps-playwright'
        ],
        browser: {
            timeout: {
                present: 10000,
                visible: 20000    
            },
            capabilities: {
                browserName: 'chromium'
            }
        },
        pageObject: new App()
    }
}

playwright steps provide a couple of additional configuration properties

NameTypeDescriptionDefault
browserobjectobject describing playwright config{}
pageObjectobjectinstance of page object definitions (page object){}
screenshotstringscreenshot strategy (beforeStep, afterStep, onFail)

Parameter Typesâ€‹

playwrightConditionWaitâ€‹

condition of element to wait (can be negated with not)

  • to be visible
  • to be present
  • to be invisible

playwrightValueWaitâ€‹

condition of value to wait (can be negated with not)

  • to be equal
  • to contain
  • to be above
  • to be below

playwrightValidationâ€‹

validation of values (can be negated with not)

  • to be equal
  • to be strictly equal
  • to be deeply equal
  • to have member
  • to be match
  • to contain
  • to be above
  • to be below
  • to be greater than
  • to be less than
  • to have type
